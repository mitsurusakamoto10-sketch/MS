# PLAUD NOTE 振り返りワークフロー（利用ガイド）

PLAUD NOTE の録音を、Claude との対話で議事録（md ファイル）に整え、
Google Drive に蓄積するためのワークフローです。

## 目的

1. **知識蓄積** — 日々の会話の議事を知識として蓄積し、いつでも振り返れるようにする
2. **話し方フィードバック** — 文字起こしをもとに、自分の話し方の癖や言葉遣いへの
   フィードバックを AI から受け、ビジネススキルを高める
3. **仕事の記録** — 指示を受けた仕事・依頼した仕事を記録し、誤解を防ぐ
4. **議事録作成** — 定期的に必要となる会議の議事録を作成する

※ 目的が変わったら `CLAUDE.md` と `.claude/skills/furikaeri/SKILL.md` を
更新すれば、以後の運用に反映されます。

## 使い方

### 振り返り（議事録化）

Claude Code のチャットで **「振り返り」** と入力するだけです。

1. Claude が PLAUD NOTE からまだ md 化していない録音を探します
2. 1件ずつ、内容を確認しながら「いつ・誰と・どんな場の会話か」
   「どの話者が自分か」をチャットで聞いてきます
3. 回答すると、議事録 md と話し方フィードバック md が生成され、
   Google Drive の「mdファイル」フォルダに自動保存されます

### 過去の内容を調べる（ナレッジQ&A）

チャットで普通に質問してください（例:「先週の◯◯さんとの電話で決まったことは？」）。
Claude が Drive 内の議事録を検索して、出典付きで答えます。

## 保存先と命名規約

| 種類 | 保存先 | ファイル名 |
|---|---|---|
| 議事録 | Drive「mdファイル」直下 | `YYYY-MM-DD_相手_タイトル.md` |
| 話し方フィードバック | Drive「mdファイル/話し方フィードバック/」 | `YYYY-MM-DD_タイトル.md` |

## 前提条件

- Claude のセッションに **PLAUD NOTE** と **Google Drive** のコネクタ（MCP）が
  接続されていること
- MCP ツールの呼び出しには承認を求められることがあります

## 注意事項

- 議事録・文字起こしの内容は Google Drive のみに保存されます。
  この GitHub リポジトリには手順だけを置き、会話の内容は一切コミットしません
- 公開ポータル（index.html）には議事内容を表示しません

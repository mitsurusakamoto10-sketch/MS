# マイポータル（静的サイト版）

自分用ポータルサイトのトップページです。
**素のHTML / CSS / JavaScript** だけで作られており、ビルド不要で
そのまま **Cloudflare Pages** に公開できます。

外部APIは呼び出さず、表示内容はすべて **サンプル（架空）データ** です。
社内情報・個人情報・機密情報は含めていません。

## ファイル構成

```
index.html   … ページの骨組み（ヘッダー + 6カードの枠）
style.css    … デザイン・レスポンシブ（スマホ→PCで列数が変化）
script.js    … サンプルデータの表示・ToDoの完了切替
```

## カード一覧

1. 今日のToDo（クリックで完了の切替）
2. メモ・備忘録
3. 天気予報
4. プレスリリース更新
5. EDINET / 上場REIT物件取引チェック
6. MLBドジャース速報

## ローカルでの確認手順

ビルドは不要です。以下のいずれかの方法で確認できます。

### 方法A：ファイルを直接開く（一番かんたん）

`index.html` をダブルクリックしてブラウザで開くだけです。

### 方法B：簡易サーバーで開く（本番に近い形・推奨）

ターミナルでこのフォルダに移動し、次のどちらかを実行します。

```bash
# Python が入っている場合
python3 -m http.server 8000

# Node.js が入っている場合
npx serve .
```

表示されたアドレス（例：`http://localhost:8000`）をブラウザで開きます。
止めるときはターミナルで `Ctrl + C` です。

## 表示内容の変え方

カードに出る文字や数字は、すべて `script.js` の上部（サンプルデータ）に
まとまっています。そこを書き換えると画面の表示も変わります。

## Cloudflare Pages への公開

ビルドコマンドは不要です。Cloudflare Pages の設定で
- ビルドコマンド：（空欄）
- 出力ディレクトリ：`/`（ルート）

としてこのリポジトリを接続すれば、`index.html` がそのまま公開されます。

# マイポータル（初期版）

自分用ポータルサイトの「見た目だけ」の初期版です。
外部APIは呼び出さず、すべて**サンプル（架空）データ**を表示します。

## 技術構成

- Next.js 14（App Router）
- TypeScript
- Tailwind CSS v3
- 1ページ構成・日本語UI

## カード一覧

1. 今日のToDo
2. メモ・備忘録
3. 天気予報
4. プレスリリース更新
5. EDINET / 上場REIT物件取引チェック
6. MLBドジャース速報

## 使い方

```bash
# 1回だけ：必要なライブラリをインストール
npm install

# 開発サーバーを起動（http://localhost:3000 を開く）
npm run dev

# 本番用にビルドしてエラーがないか確認
npm run build
```

## 表示内容の変え方

カードに出ている文字や数字は、すべて `lib/sampleData.ts` にまとまっています。
このファイルの値を書き換えると、画面の表示も変わります。

## ファイル構成

```
app/
  layout.tsx        … 全ページ共通の枠（タイトル・フォントなど）
  page.tsx          … トップページ（カードを並べる）
  globals.css       … 全体のスタイル
components/
  Card.tsx          … カードの共通の見た目（枠）
  cards/            … 6種類のカード本体
lib/
  sampleData.ts     … 表示用のサンプルデータ（ここを編集）
```

> 注意：本物の業務データは含めていません。実データを扱う場合は別途ご相談ください。
